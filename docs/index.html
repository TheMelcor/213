A rogue like inspired game

About

This is an open source project created by five Information Technology students. We have created a desktop game inspired by the rogue like "genre". To create the functionality in the game we used Unity, and models/environment were made using 3DMax. All files are in open format.
The main focus of the game is to advance from one level/environment to another by the help of different winning conditions. The winning conditions tells something about what types of actions is required to advance to the next level. Conditions may wary from level to level.
This is a module-based games, meaning it should be easy to add and alter any material within the game, both in terms of assets and regarding the rules (winning conditions/scenarios).
